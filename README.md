Mon site vitrine — template minimal React + Vite

Structure fournie :

mon-site-vitrine/
├── public/
├── src/
├── package.json
└── vite.config.js

Pour démarrer :

```bash
npm install
npm run dev
```

Ajoute tes images dans `public/images/` (logo.png, hero.jpg) et `src/assets/images/` (profil.jpg).
